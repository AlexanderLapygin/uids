## Installing UIDs

To use UIDs modules, all you need to do is install the `@uids/core` package:

```sh
$ yarn add @uids/core

# or

$ npm install @uids/core
```

## Usage

At the moment, the library exposes such modules:

- `uids`

For example, you can use the `uids` module like this:

```ts
import { UIDs } from '@uids/core'

const uids = new UIDs()
// ...
```
